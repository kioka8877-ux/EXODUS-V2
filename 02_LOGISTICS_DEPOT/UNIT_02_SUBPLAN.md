# SOUS-PLAN TECHNIQUE — UNITÉ 02: LOGISTICS DEPOT

## Mission
Assembler les personnages Roblox avec leurs animations et props pour produire des fichiers Alembic prêts pour le rendu.

## Stack Technique (Prévu)
- Blender 4.x
- Blender Python API (bpy)
- Alembic export (.abc)

## Inputs
- `IN_ANIMATION/animation_*.pkl` — Données de pose
- `IN_ANIMATION/motion_*.bvh` — Motion capture
- Assets Roblox (personnages, props)

## Outputs
- `OUT_ALEMBIC/actor_*.abc` — Personnages animés format Alembic
- `OUT_ALEMBIC/props_*.abc` — Props animés
